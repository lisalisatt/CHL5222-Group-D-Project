# CHL5222-Group-D-Project
Analysis of a randomized controlled trial evaluating the effectiveness of computerized cognitive behavioural therapy (BtheB) on depression severity using linear mixed-effects models in R.

## Overview
This project analyzes data from a randomized controlled trial evaluating the effectiveness of computerized cognitive behavioural therapy (BtheB) compared to treatment as usual (TAU) in reducing depression severity.

Depression severity is measured using the Beck Depression Inventory (BDI) across multiple time points. The analysis uses linear mixed-effects models to assess treatment effects during both active treatment and post-treatment follow-up.

## Repository Structure
- `Group D Report.qmd` : Main Quarto file containing data analysis and report
- `Group D Report.pdf` : Final rendered report
- `btheb.txt` : Dataset used for analysis
